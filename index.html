Q.1 What are the different data types present in javascript?

 1. Primitive data types:
    a. Number     eg. let num = 12;
    b. String     eg. let str = "Assignment-1";
    c. Boolean    eg. let bool = true;
    d. Bigint     eg. let bigNum = BigInt("12345678355477322827366");
    e. Null       eg. let str2 = null; 
    f. Undefined  eg. let y;
    g. Symbol     eg. let symbol = Symbol("Hi!")

2. Non-primitive data types:
   a. Array      eg. let arr = [1, 2, 3, 4];
   b. Objects    eg. let values = {a: 23, b: 10};
